# Rock Paper Scissors
A simple console-based implementation of the classic Rock–Paper–Scissors game. The player selects a move, the computer randomly chooses its own move, and the game determines the winner. This project is beginner-friendly and demonstrates essential C++ programming concepts such as loops, conditions, functions, and randomness.

# Game play
Choose between Rock, Paper, or Scissors

The computer randomly selects its move

The winner is determined using standard game rules

You can play as many rounds as you want

# Features
Clean and readable C++ code

Randomized computer choices

Input validation

Replay option

Simple function for mapping numeric choices to names
